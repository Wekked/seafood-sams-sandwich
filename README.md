# Seafood Sam's — Inventory Tracker (Sandwich, MA)

Cloud-synced inventory management for Seafood Sam's Sandwich location.

## Features
- 540 items tracked across 10 storage locations
- Real-time sync across all devices via Supabase
- Drag-and-drop walkthrough reordering
- Offline support (works in freezers/basements)
- iPad-optimized with PWA support
- CSV export
- Food cost calculator

## Tech Stack
- React 18 (CDN, no build tools)
- Supabase (PostgreSQL + Auth + Realtime)
- GitHub Pages hosting
